Final cleanup

Filtering out 0 value cases and data set to the year 2022. Additionally pivot using group by to get sum of cases and deaths by month and year

#filter out 0 cases
global <- global %>% filter(cases >0)
global <- global %>%filter(year(date) == 2022)
global$monthyear <- format(ymd(global$date), "%b %Y")

pivot <- global %>%
    group_by(monthyear) %>%
    summarise(cases = sum(cases),
        deaths = sum(deaths))

Analysis

For the first and second visual I went with a time lapse of cases and deaths in two separate charts over the last year. Blue is for cases, Red is for deaths. As you can see from the charts they both follow a similar shape, leading me to believe theres a linear relationship that could be modeled. An interesting note is that there was a decline in both around February of last year. This is most likely from two things. The first being the introduction of another varient. This can cause fear and resulting people to isolate and prevent the spread of the virus, thus causing less deaths. Secondly would have to do with weather, it seems that during colder months, people spend more time alone inside versus going to to see friends or family. I believe this also had an effect on the reduction of both cases and deaths. Additionally, when testing, I had the two charts overlapping to see if there was a point where they would meet, however I did not see that in the visual, as the number of cases greatly outnumbers the total deaths.

For the final visual I went for a bar graph to get a better picture of how each state is doing in terms of recent deaths in the US. From the graph we can see that California, Texas, Georgia and New York have the highest amount of total deaths in last month, with California being the highest. These are considered highly populous states, so its understandable that the likelyhood of infection and then death are more likely.

Model

Predicts Number of deaths, given number of cases

model <- lm(deaths ~ cases, data = global)

pred <- predict(model)

global$predicted <- pred

ggplot(data.frame(actual_X = global$cases, actual_Y =  global$deaths, predicted = pred),
    aes(x = cases, y = deaths))+
    geom_point(aes(x =actual_X, y = actual_Y), color = "blue") +
    geom_point(aes(x =actual_X, y=pred), color ="red")

The model for predicting global deaths based on the number of global cases appears to be linear in fashion. Generally speaking, given a number of cases, one can predict the expected number of deaths

Bias

Overall, I would say my analysis does not have an inherent bias, as I included all countries and states into my two analysis. I did this intentionally so there would be less likely to focus on one area and then mistakenly conclude that it represents the entire population. The only bias I could conceive would be if one population is reporting more than another, which would skew the datasets.
